Sales Data Input : 
This program prompts for units sold and price to calculate total revenue. It uses a reusable function and a while loop to catch ValueError exceptions for non-numeric entries.

Inventory Quantity Checker: 
This script compares current stock against a reorder threshold to alert managers when to restock. It includes error handling for non-integers and prevents crashes from ZeroDivisionError during percentage calculations.

Customer Age Verification: 
This tool verifies if a customer is 18 or older for marketing eligibility. It demonstrates handling ValueError for age inputs and uses a NameError simulation to show how to recover from variable reference issues.

Financial Ratio Calculator: 
This calculator determines the profit margin ratio by dividing profit by revenue. It utilizes a try-except-else block to handle division by zero and ensures valid float inputs through defensive programming.

Employee Salary Adjustment : 
This HR-focused script calculates new salaries based on a percentage increase. It uses a helper function for validation and includes logic to prevent invalid negative percentage adjustments.
